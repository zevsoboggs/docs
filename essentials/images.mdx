## Изображение

### Используя Markdown

[Синтаксис markdown](https://www.markdownguide.org/basic-syntax/#images) позволяет добавлять изображения с помощью следующего кода

```md
![заголовок](/path/image.jpg)
```

Обратите внимание, что размер файла изображения должен быть менее 5MB. В противном случае рекомендуем использовать хостинг типа [Cloudinary](https://cloudinary.com/) или [S3](https://aws.amazon.com/s3/). Затем можно использовать этот URL для встраивания.

### Используя встраивание

Для большей гибкости с изображениями можно также использовать [встраивание](/writing-content/embed)

```html
<img height="200" src="/path/image.jpg" />
```

<Tip>

Mintlify поддерживает [HTML теги в Markdown](https://www.markdownguide.org/basic-syntax/#html). Это полезно если вы предпочитаете HTML теги синтаксису Markdown, и позволяет создавать документацию с бесконечной гибкостью.

</Tip>

### iFrames

Загружает другую HTML страницу внутри документа. Чаще всего используется для встраивания видео.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
