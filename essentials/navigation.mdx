Меню навигации — это список ссылок на каждом сайте.

Скорее всего вы будете обновлять `docs.json` каждый раз при добавлении новой страницы. Страницы не появляются автоматически.

## Синтаксис навигации

Наш синтаксис навигации рекурсивный, что означает возможность создания вложенных групп навигации. Не нужно включать `.mdx` в названиях страниц.

<CodeGroup>

```json Обычная навигация
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": ["quickstart"]
        }
      ]
    }
  ]
}
```

```json Вложенная навигация
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Getting Started",
          "pages": [
            "quickstart",
            {
              "group": "Nested Reference Pages",
              "pages": ["nested-reference-page"]
            }
          ]
        }
      ]
    }
  ]
}
```

</CodeGroup>

## Папки

Просто поместите MDX файлы в папки и обновите пути в `docs.json`.

Например, чтобы страница была доступна по адресу `https://yoursite.com/your-folder/your-page`, создайте папку `your-folder` с MDX файлом `your-page.mdx`.

<Warning>

Нельзя использовать `api` как название папки верхнего уровня. Mintlify использует Next.js, который резервирует папку `api` для внутренних серверных вызовов. Название папки типа `api-reference` будет работать.

</Warning>

```json Навигация с папкой
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

## Скрытые страницы

MDX файлы, не включённые в `docs.json`, не будут отображаться в боковой панели, но доступны через поиск и по прямой ссылке.
