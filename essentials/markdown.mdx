## Заголовки

Лучше всего подходят для заголовков секций.

```md
## Заголовки
```

### Подзаголовки

Лучше всего подходят для заголовков подсекций.

```md
### Подзаголовки
```

<Tip>

Каждый **заголовок** и **подзаголовок** создаёт якорь и отображается в оглавлении справа.

</Tip>

## Форматирование текста

Мы поддерживаем большинство форматирования markdown. Просто добавьте `**`, `_` или `~` вокруг текста для форматирования.

| Стиль         | Как написать      | Результат       |
| ------------- | ----------------- | --------------- |
| Жирный        | `**жирный**`      | **жирный**      |
| Курсив        | `_курсив_`        | _курсив_        |
| Зачёркнутый   | `~зачёркнутый~`   | ~зачёркнутый~   |

Можно комбинировать. Например, напишите `**_жирный и курсив_**` чтобы получить **_жирный и курсив_** текст.

Для надстрочного и подстрочного текста нужно использовать HTML. Добавьте `<sup>` или `<sub>` вокруг текста.

| Размер текста | Как написать             | Результат              |
| ------------- | ------------------------ | ---------------------- |
| Надстрочный   | `<sup>надстрочный</sup>` | <sup>надстрочный</sup> |
| Подстрочный   | `<sub>подстрочный</sub>` | <sub>подстрочный</sub> |

## Ссылки на страницы

Можно добавить ссылку, обернув текст в `[]()`. Напишите `[ссылка на google](https://google.com)` для [ссылки на google](https://google.com).

Ссылки на страницы в вашей документации должны быть относительными от корня. То есть нужно включать полный путь папки. Например, `[ссылка на текст](/writing-content/text)` ссылается на страницу "Text" в секции компонентов.

Относительные ссылки типа `[ссылка на текст](../text)` будут открываться медленнее, потому что мы не можем оптимизировать их так же легко.

## Цитаты

### Однострочные

Чтобы создать цитату, добавьте `>` перед абзацем.

> Дороти следовала за ней через множество красивых комнат её замка.

```md
> Дороти следовала за ней через множество красивых комнат её замка.
```

### Многострочные

> Дороти следовала за ней через множество красивых комнат её замка.
>
> Ведьма заставила её чистить кастрюли и чайники, подметать пол и поддерживать огонь в печи.

```md
> Дороти следовала за ней через множество красивых комнат её замка.
>
> Ведьма заставила её чистить кастрюли и чайники, подметать пол и поддерживать огонь в печи.
```

### LaTeX

Mintlify поддерживает [LaTeX](https://www.latex-project.org) через компонент Latex.

<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>

```md
<Latex>8 x (vk x H1 - H2) = (0,1)</Latex>
```
