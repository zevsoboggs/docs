## MCP Сервер для ИИ-агентов

<Info>
**Статус:** В разработке
</Info>

Мы создаём **MCP (Model Context Protocol) сервер** для интеграции Love&Pay с ИИ-агентами.

### Что это даёт?

<CardGroup cols={2}>
  <Card title="Автоматизация" icon="wand-magic-sparkles">
    ИИ-агенты смогут автоматически создавать счета, проверять статусы и управлять платежами
  </Card>
  <Card title="Интеграция с Claude" icon="message-bot">
    Работайте с Love&Pay прямо из Claude Desktop, Cursor или других MCP-клиентов
  </Card>
  <Card title="Голосовые команды" icon="microphone">
    Создавайте счета и проверяйте платежи голосовыми командами через ИИ-ассистентов
  </Card>
  <Card title="Умные воркфлоу" icon="diagram-project">
    Стройте сложные бизнес-процессы с помощью ИИ-агентов
  </Card>
</CardGroup>

### Примеры использования

```
Пользователь: Создай счёт на 5000 рублей для клиента ivan@example.com

ИИ-агент: Создал счёт #INV-12345 на 5000 RUB.
Ссылка для оплаты: https://pay.loveandpay.io/inv-12345
```

```
Пользователь: Покажи статистику платежей за сегодня

ИИ-агент: За сегодня:
- Создано счетов: 47
- Оплачено: 38 (80.8%)
- Сумма: 156 400 RUB
```

## Приём криптовалюты

<Check>
**Статус:** Доступно!
</Check>

Для мерчантов уже доступен **приём платежей в криптовалюте** через LoveCrypto API.

### Возможности

<CardGroup cols={2}>
  <Card title="USDT и USDC" icon="coins">
    На сетях TRC20, ERC20, BEP20
  </Card>
  <Card title="Низкие комиссии" icon="percent">
    Всего 1% на приём и вывод
  </Card>
  <Card title="Простой API" icon="code">
    REST API с HMAC подписью
  </Card>
  <Card title="Вебхуки" icon="bell">
    Уведомления о платежах в реальном времени
  </Card>
</CardGroup>

### Преимущества для мерчантов

- **Глобальные платежи** — принимайте оплату от клиентов из любой точки мира
- **Низкие комиссии** — конкурентные ставки на криптовалютные транзакции
- **Быстрые расчёты** — моментальное зачисление средств
- **Без чарджбеков** — криптовалютные платежи необратимы

### Как это будет работать

<Steps>
  <Step title="Включите криптоплатежи">
    Активируйте приём криптовалюты в панели управления
  </Step>
  <Step title="Создайте счёт">
    Используйте тот же API с параметром `paymentMethod: 'crypto'`
  </Step>
  <Step title="Клиент оплачивает">
    Клиент выбирает криптовалюту и отправляет платёж
  </Step>
  <Step title="Получите уведомление">
    Вебхук `invoice.paid` сообщит об успешной оплате
  </Step>
</Steps>

```bash
# Создание криптовалютного счёта
curl -X POST "https://crypto.api.loveandpay.io/api/v1/invoices" \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: lc_ваш_ключ" \
  -H "X-Timestamp: 1706400000" \
  -H "X-Signature: ваша_подпись" \
  -d '{"amount": 100, "currency": "USDT_TRC20"}'
```
