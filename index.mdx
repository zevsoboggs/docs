## Что такое Love&Pay?

**Love&Pay** — платёжная платформа для российского и международного бизнеса. Принимаем оплаты через СБП (P2P-перевод по QR), карты, криптовалюты. Встроенный KYC, антифрод, API v2 с HMAC-подписью, вебхуки, реферальная программа подпартнёров.

## Жизненный цикл счёта (важно!)

<Note>
**Внутренний счёт Love&Pay живёт 1 час**, но СБП-QR от банка действителен только **5 минут**.
Внутри часа мы **автоматически пересоздаём QR каждые 5 минут** — клиент всегда видит свежий код. От вас никаких действий не требуется.
</Note>

Подробности — в [Жизненном цикле счёта](/guides/invoice-lifecycle).

## Быстрый старт за 4 шага

<Steps>
  <Step title="Войдите в кабинет">
    Откройте [Панель управления](https://loveandpay.io/dashboard).
    Регистрация — по приглашению или через реферальную ссылку партнёра.
  </Step>
  <Step title="Получите API-ключи">
    Перейдите в **Профиль → API** ([прямая ссылка](https://loveandpay.io/dashboard/profile)).
    Скопируйте **API Key** (постоянный) и нажмите **«Сгенерировать секретный ключ»** для API v2 с HMAC-подписью.
  </Step>
  <Step title="Интегрируйте API">
    Используйте [API v2](/api-reference/v2/introduction) — рекомендуемая версия с HMAC-SHA256.
    Минимальный запрос — [Создать счёт](/api-reference/v2/invoices/create).
  </Step>
  <Step title="Настройте вебхуки">
    Зайдите в **[Вебхуки](https://loveandpay.io/dashboard/webhooks)** в кабинете и создайте подписку на события (`invoice.paid`, `invoice.expired`, `invoice.refunded`).
    <Warning>Вебхуки настраиваются **только через веб-кабинет** — API для создания вебхуков нет.</Warning>
  </Step>
</Steps>
