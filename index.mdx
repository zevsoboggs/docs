## Что такое Love&Pay?

Love&Pay — это платёжная система для приёма платежей через Систему Быстрых Платежей (СБП). Мы предоставляем простой и удобный способ принимать оплату от ваших клиентов.

## Быстрый старт

<Steps>
  <Step title="Зарегистрируйтесь">
    Создайте аккаунт в [панели управления](https://loveandpay.io/dashboard)
  </Step>
  <Step title="Получите API ключ">
    Перейдите в **Настройки → API ключи** и создайте новый ключ
  </Step>
  <Step title="Интегрируйте API">
    Используйте наш [API](/api-reference/introduction) для создания счетов
  </Step>
  <Step title="Принимайте платежи">
    Отправляйте клиентам ссылки на оплату или QR-коды
  </Step>
</Steps>
