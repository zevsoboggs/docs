## Что такое Identy?

Identy — это сервис валидации подписей на основе искусственного интеллекта. Он анализирует изображения подписей и определяет их подлинность.

## Поддерживаемые форматы

| Формат | MIME-тип |
|--------|----------|
| PNG | `image/png` |
| JPEG | `image/jpeg` |
| GIF | `image/gif` |
| WebP | `image/webp` |

<Note>
  Максимальный размер файла: **10 MB**
</Note>

## Категории подписей

AI классифицирует подписи по следующим категориям:

| Код | Категория | Описание |
|-----|-----------|----------|
| 0 | `Unknown` | Не удалось классифицировать |
| 1 | `ValidSignature` | Подлинная подпись |
| 2 | `InvalidSignature` | Недействительная подпись |
| 3 | `Scribble` | Каракули, неразборчиво |
| 4 | `PrintedText` | Печатный текст вместо подписи |
| 5 | `Drawing` | Рисунок |
| 6 | `BlankOrEmpty` | Пустое изображение |
| 7 | `LowQuality` | Низкое качество изображения |
| 8 | `PartialSignature` | Частичная/обрезанная подпись |

## Быстрый старт

<Steps>
  <Step title="Подготовьте изображение">
    Сделайте фото или скан подписи в формате PNG/JPEG
  </Step>
  <Step title="Отправьте на валидацию">
    Используйте эндпоинт `/api/Signature/validate/upload` для загрузки файла
  </Step>
  <Step title="Получите результат">
    AI вернёт оценку подлинности, категорию и рекомендации
  </Step>
</Steps>
