---

<Note>
Чеки используются для anti-fraud проверки платежей. Клиент загружает фото чека, система автоматически проверяет сумму и дату.
</Note>

## Заголовки

<ParamField header="x-api-key" type="string" required>
  Ваш API ключ
</ParamField>

<ParamField header="x-timestamp" type="string" required>
  Unix timestamp в миллисекундах
</ParamField>

<ParamField header="x-signature" type="string" required>
  HMAC-SHA256 подпись
</ParamField>

## Query параметры

<ParamField query="limit" type="number" default="50">
  Количество записей (максимум 100)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Смещение для пагинации
</ParamField>

<ParamField query="status" type="string">
  Фильтр по статусу: `PENDING`, `APPROVED`, `REJECTED`
</ParamField>

<ParamField query="invoiceId" type="string">
  Фильтр по UUID счёта
</ParamField>

```javascript JavaScript
const crypto = require('crypto');

const API_KEY = 'pk_live_xxx';
const SECRET_KEY = 'sk_live_xxx';

const timestamp = Date.now().toString();
const method = 'GET';
const path = '/api/v2/receipts';

const bodyHash = crypto.createHash('sha256').update('').digest('hex');
const message = `${method}${path}${timestamp}${bodyHash}`;
const signature = crypto.createHmac('sha256', SECRET_KEY).update(message).digest('hex');

const response = await fetch('https://loveandpay.io' + path + '?status=APPROVED', {
  method,
  headers: {
    'x-api-key': API_KEY,
    'x-timestamp': timestamp,
    'x-signature': signature
  }
});

console.log(await response.json());
```

```python Python
import hashlib
import hmac
import time
import requests

API_KEY = 'pk_live_xxx'
SECRET_KEY = 'sk_live_xxx'

timestamp = str(int(time.time() * 1000))
method = 'GET'
path = '/api/v2/receipts'

body_hash = hashlib.sha256(''.encode()).hexdigest()
message = f"{method}{path}{timestamp}{body_hash}"
signature = hmac.new(SECRET_KEY.encode(), message.encode(), hashlib.sha256).hexdigest()

response = requests.get(
    'https://loveandpay.io' + path,
    params={'status': 'APPROVED'},
    headers={
        'x-api-key': API_KEY,
        'x-timestamp': timestamp,
        'x-signature': signature
    }
)

print(response.json())
```
</RequestExample>

## Статусы чеков

| Статус | Описание |
|--------|----------|
| `PENDING` | Ожидает проверки |
| `APPROVED` | Чек подтверждён — сумма и дата совпадают |
| `REJECTED` | Чек отклонён — данные не совпадают или нечитаемы |
