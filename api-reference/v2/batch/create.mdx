</ParamField>

<RequestExample>
```bash cURL
TIMESTAMP=$(date +%s%3N)
BODY_HASH=$(echo -n "" | sha256sum | cut -d' ' -f1)
MESSAGE="POST/api/v2/batch/invoices${TIMESTAMP}${BODY_HASH}"
SIGNATURE=$(echo -n "$MESSAGE" | openssl dgst -sha256 -hmac "sk_live_xxx" | cut -d' ' -f2)

curl -X POST "https://loveandpay.io/api/v2/batch/invoices" \
  -H "x-api-key: pk_live_xxx" \
  -H "x-timestamp: ${TIMESTAMP}" \
  -H "x-signature: ${SIGNATURE}" \
  -F "file=@invoices.csv" \
  -F "format=csv"
```

```javascript JavaScript
const crypto = require('crypto');
const FormData = require('form-data');
const fs = require('fs');

const API_KEY = 'pk_live_xxx';
const SECRET_KEY = 'sk_live_xxx';

const timestamp = Date.now().toString();
const method = 'POST';
const path = '/api/v2/batch/invoices';

// Для multipart запросов body = пустая строка
const bodyHash = crypto.createHash('sha256').update('').digest('hex');
const message = `${method}${path}${timestamp}${bodyHash}`;
const signature = crypto.createHmac('sha256', SECRET_KEY).update(message).digest('hex');

const form = new FormData();
form.append('file', fs.createReadStream('invoices.csv'));
form.append('format', 'csv');

const response = await fetch('https://loveandpay.io' + path, {
  method,
  headers: {
    'x-api-key': API_KEY,
    'x-timestamp': timestamp,
    'x-signature': signature,
    ...form.getHeaders()
  },
  body: form
});

const result = await response.json();

console.log(`Создано: ${result.summary.successful}/${result.summary.total}`);
console.log(`Ошибок: ${result.summary.failed}`);

// Вывести ошибки
result.results
  .filter(r => !r.success)
  .forEach(r => console.log(`Строка ${r.row}: ${r.error}`));
```

```python Python
import hashlib
import hmac
import time
import requests

API_KEY = 'pk_live_xxx'
SECRET_KEY = 'sk_live_xxx'

timestamp = str(int(time.time() * 1000))
method = 'POST'
path = '/api/v2/batch/invoices'

# Для multipart запросов body = пустая строка
body_hash = hashlib.sha256(''.encode()).hexdigest()
message = f"{method}{path}{timestamp}{body_hash}"
signature = hmac.new(SECRET_KEY.encode(), message.encode(), hashlib.sha256).hexdigest()

with open('invoices.csv', 'rb') as f:
    response = requests.post(
        'https://loveandpay.io' + path,
        headers={
            'x-api-key': API_KEY,
            'x-timestamp': timestamp,
            'x-signature': signature
        },
        files={'file': f},
        data={'format': 'csv'}
    )

result = response.json()

print(f"Создано: {result['summary']['successful']}/{result['summary']['total']}")
print(f"Ошибок: {result['summary']['failed']}")

# Вывести ошибки
for r in result['results']:
    if not r['success']:
        print(f"Строка {r['row']}: {r['error']}")
```

## Ограничения

| Параметр | Значение |
|----------|----------|
| Максимум записей | 1000 |
| Максимум размер файла | 5MB |
| Поддерживаемые форматы | CSV, JSON |

<Note>
Для multipart/form-data запросов при вычислении HMAC подписи используйте **пустую строку** как тело запроса.
</Note>
