<Warning>
**Удаление вебхуков через публичный API отключено** — только через веб-кабинет.
</Warning>

## Как удалить

[Панель управления → Вебхуки](https://loveandpay.io/dashboard/webhooks) → карточка нужного вебхука → **Удалить** (или **Отключить** если хотите оставить историю доставки).

Полное руководство — [Вебхуки](/guides/webhooks).
