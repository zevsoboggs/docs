## Заголовки

<ParamField header="x-api-key" type="string" required>
  Ваш API ключ
</ParamField>

<ParamField header="x-timestamp" type="string" required>
  Unix timestamp в миллисекундах
</ParamField>

<ParamField header="x-signature" type="string" required>
  HMAC-SHA256 подпись
</ParamField>

## Тело запроса

<ParamField body="url" type="string" required>
  URL для отправки событий (HTTPS обязателен)
</ParamField>

<ParamField body="events" type="array" required>
  Список событий для подписки. Доступные события:
  - `invoice.created` — счёт создан
  - `invoice.paid` — счёт оплачен
  - `invoice.expired` — срок счёта истёк
  - `invoice.cancelled` — счёт отменён
  - `kyc.approved` — KYC верификация пройдена
  - `kyc.declined` — KYC верификация отклонена
  - `receipt.approved` — чек подтверждён
  - `receipt.rejected` — чек отклонён
</ParamField>

```javascript JavaScript
const crypto = require('crypto');

const API_KEY = 'pk_live_xxx';
const SECRET_KEY = 'sk_live_xxx';

const timestamp = Date.now().toString();
const method = 'POST';
const path = '/api/v2/webhooks';
const body = JSON.stringify({
  url: 'https://your-site.com/webhooks/payments',
  events: ['invoice.paid', 'invoice.expired']
});

const bodyHash = crypto.createHash('sha256').update(body).digest('hex');
const message = `${method}${path}${timestamp}${bodyHash}`;
const signature = crypto.createHmac('sha256', SECRET_KEY).update(message).digest('hex');

const response = await fetch('https://loveandpay.io' + path, {
  method,
  headers: {
    'Content-Type': 'application/json',
    'x-api-key': API_KEY,
    'x-timestamp': timestamp,
    'x-signature': signature
  },
  body
});

const { webhook } = await response.json();

// ВАЖНО: Сохраните secretKey!
console.log('Секретный ключ вебхука:', webhook.secretKey);
```

```python Python
import hashlib
import hmac
import time
import requests
import json

API_KEY = 'pk_live_xxx'
SECRET_KEY = 'sk_live_xxx'

timestamp = str(int(time.time() * 1000))
method = 'POST'
path = '/api/v2/webhooks'
body = json.dumps({
    'url': 'https://your-site.com/webhooks/payments',
    'events': ['invoice.paid', 'invoice.expired']
})

body_hash = hashlib.sha256(body.encode()).hexdigest()
message = f"{method}{path}{timestamp}{body_hash}"
signature = hmac.new(SECRET_KEY.encode(), message.encode(), hashlib.sha256).hexdigest()

response = requests.post(
    'https://loveandpay.io' + path,
    headers={
        'Content-Type': 'application/json',
        'x-api-key': API_KEY,
        'x-timestamp': timestamp,
        'x-signature': signature
    },
    data=body
)

data = response.json()
# ВАЖНО: Сохраните secretKey!
print(f"Секретный ключ вебхука: {data['webhook']['secretKey']}")
```
