## Заголовки

<ParamField header="x-api-key" type="string" required>
  Ваш API ключ
</ParamField>

<ParamField header="x-timestamp" type="string" required>
  Unix timestamp в миллисекундах
</ParamField>

<ParamField header="x-signature" type="string" required>
  HMAC-SHA256 подпись
</ParamField>

## Query параметры

<ParamField query="limit" type="number" default="50">
  Количество записей (максимум 100)
</ParamField>

<ParamField query="offset" type="number" default="0">
  Смещение для пагинации
</ParamField>

<ParamField query="search" type="string">
  Поиск по имени, email или телефону
</ParamField>

<ParamField query="kycStatus" type="string">
  Фильтр по KYC статусу: `PENDING`, `VERIFIED`, `REJECTED`
</ParamField>

<ParamField query="trustLevel" type="string">
  Фильтр по уровню доверия: `NEW`, `LEVEL_1`, `LEVEL_2`, `LEVEL_3`
</ParamField>

```javascript JavaScript
const crypto = require('crypto');

const API_KEY = 'pk_live_xxx';
const SECRET_KEY = 'sk_live_xxx';

const timestamp = Date.now().toString();
const method = 'GET';
const path = '/api/v2/customers';

const bodyHash = crypto.createHash('sha256').update('').digest('hex');
const message = `${method}${path}${timestamp}${bodyHash}`;
const signature = crypto.createHmac('sha256', SECRET_KEY).update(message).digest('hex');

const response = await fetch('https://loveandpay.io' + path + '?limit=50&kycStatus=VERIFIED', {
  method,
  headers: {
    'x-api-key': API_KEY,
    'x-timestamp': timestamp,
    'x-signature': signature
  }
});

console.log(await response.json());
```

```python Python
import hashlib
import hmac
import time
import requests

API_KEY = 'pk_live_xxx'
SECRET_KEY = 'sk_live_xxx'

timestamp = str(int(time.time() * 1000))
method = 'GET'
path = '/api/v2/customers'

body_hash = hashlib.sha256(''.encode()).hexdigest()
message = f"{method}{path}{timestamp}{body_hash}"
signature = hmac.new(SECRET_KEY.encode(), message.encode(), hashlib.sha256).hexdigest()

response = requests.get(
    'https://loveandpay.io' + path,
    params={'limit': 50, 'kycStatus': 'VERIFIED'},
    headers={
        'x-api-key': API_KEY,
        'x-timestamp': timestamp,
        'x-signature': signature
    }
)

print(response.json())
```

## Уровни доверия

| Уровень | Описание |
|---------|----------|
| `NEW` | Новый клиент |
| `LEVEL_1` | Базовый уровень (1-2 успешных платежа) |
| `LEVEL_2` | Средний уровень (3-5 платежей или KYC) |
| `LEVEL_3` | Высокий уровень (6+ платежей и KYC) |
