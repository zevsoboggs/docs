## Заголовки

<ParamField header="x-api-key" type="string" required>
  Ваш API ключ
</ParamField>

<ParamField header="x-timestamp" type="string" required>
  Unix timestamp в миллисекундах
</ParamField>

<ParamField header="x-signature" type="string" required>
  HMAC-SHA256 подпись
</ParamField>

## Query параметры

<ParamField query="phone" type="string">
  Телефон клиента в формате `+7XXXXXXXXXX`
</ParamField>

<ParamField query="email" type="string">
  Email клиента
</ParamField>

<Note>
Необходимо указать хотя бы один параметр: `phone` или `email`
</Note>

```javascript JavaScript
const crypto = require('crypto');

const API_KEY = 'pk_live_xxx';
const SECRET_KEY = 'sk_live_xxx';

const timestamp = Date.now().toString();
const method = 'GET';
const path = '/api/v2/customers/check';

const bodyHash = crypto.createHash('sha256').update('').digest('hex');
const message = `${method}${path}${timestamp}${bodyHash}`;
const signature = crypto.createHmac('sha256', SECRET_KEY).update(message).digest('hex');

const response = await fetch('https://loveandpay.io' + path + '?phone=' + encodeURIComponent('+79991234567'), {
  method,
  headers: {
    'x-api-key': API_KEY,
    'x-timestamp': timestamp,
    'x-signature': signature
  }
});

const data = await response.json();

if (data.exists && data.kycVerified) {
  console.log('Клиент верифицирован:', data.customer.fullName);
} else if (data.exists) {
  console.log('Клиент найден, но не верифицирован');
} else {
  console.log('Клиент не найден');
}
```

```python Python
import hashlib
import hmac
import time
import requests
from urllib.parse import quote

API_KEY = 'pk_live_xxx'
SECRET_KEY = 'sk_live_xxx'

timestamp = str(int(time.time() * 1000))
method = 'GET'
path = '/api/v2/customers/check'

body_hash = hashlib.sha256(''.encode()).hexdigest()
message = f"{method}{path}{timestamp}{body_hash}"
signature = hmac.new(SECRET_KEY.encode(), message.encode(), hashlib.sha256).hexdigest()

response = requests.get(
    'https://loveandpay.io' + path,
    params={'phone': '+79991234567'},
    headers={
        'x-api-key': API_KEY,
        'x-timestamp': timestamp,
        'x-signature': signature
    }
)

data = response.json()

if data['exists'] and data['kycVerified']:
    print(f"Клиент верифицирован: {data['customer']['fullName']}")
elif data['exists']:
    print('Клиент найден, но не верифицирован')
else:
    print('Клиент не найден')
```

## Примеры использования

### Проверка перед созданием счёта

```javascript
async function createInvoiceWithKycCheck(customerPhone, amount) {
  // 1. Проверяем клиента
  const checkResult = await checkCustomer(customerPhone);

  if (!checkResult.exists) {
    // Новый клиент — создаём счёт с KYC
    return createInvoice({
      amount,
      customerPhone,
      kycRequired: true
    });
  }

  if (!checkResult.kycVerified) {
    // Клиент есть, но не верифицирован
    return createInvoice({
      amount,
      customerPhone,
      kycRequired: true
    });
  }

  // Клиент верифицирован — счёт без KYC
  return createInvoice({
    amount,
    customerPhone,
    kycRequired: false
  });
}
```
