## Описание полей

| Поле | Описание |
|------|----------|
| `available` | Доступно для вывода (подтверждённые средства) |
| `pending` | В ожидании подтверждения блокчейном |
| `total` | Общий баланс (available + pending) |
| `totalUsdValue` | Общая сумма в USD по всем валютам |
