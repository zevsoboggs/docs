## Обзор

LoveCrypto API позволяет принимать платежи в криптовалюте (USDT, USDC) на различных блокчейн-сетях.

## Базовый URL

```
https://crypto.api.loveandpay.io
```

## Поддерживаемые валюты

| Код | Название | Сеть | Decimals |
|-----|----------|------|----------|
| `USDT_TRC20` | Tether USD | Tron | 6 |
| `USDT_ERC20` | Tether USD | Ethereum | 6 |
| `USDT_BEP20` | Tether USD | BSC | 18 |
| `USDC_ERC20` | USD Coin | Ethereum | 6 |
| `USDC_BEP20` | USD Coin | BSC | 18 |

## Аутентификация

### API ключ

Передавайте ключ в заголовке `X-Api-Key`:

```bash
curl -X GET "https://crypto.api.loveandpay.io/api/v1/merchants/profile" \
  -H "X-Api-Key: lc_H8BLURMH0Vq241DqzatqOHajKUlBQERF"
```

### HMAC подпись

Для операций создания счетов и выводов требуется HMAC-SHA256 подпись:

| Заголовок | Описание |
|-----------|----------|
| `X-Api-Key` | Ваш API ключ |
| `X-Timestamp` | Unix timestamp в секундах |
| `X-Signature` | Base64 HMAC-SHA256 подпись |

**Формат payload для подписи:**
```
{timestamp}.{HTTP_METHOD}.{path}{body}
```

<Warning>
Запрос должен быть выполнен в течение **5 минут** от указанного timestamp.
</Warning>

## Формат ответов

Все ответы возвращаются в JSON формате:

**Успешный ответ:**
```json
{
  "success": true,
  "data": { ... }
}
```

**Ошибка:**
```json
{
  "success": false,
  "message": "Описание ошибки",
  "code": "ERROR_CODE"
}
```

## Коды ответов

| Код | Описание |
|-----|----------|
| 200 | Успешный запрос |
| 201 | Ресурс создан |
| 400 | Ошибка валидации |
| 401 | Ошибка аутентификации |
| 404 | Ресурс не найден |
| 429 | Превышен лимит запросов |
| 500 | Внутренняя ошибка |

## Лимиты запросов

| Операция | Лимит |
|----------|-------|
| Общий | 100 запросов/мин |
| Создание счетов | 30 запросов/мин |
| Выводы | 10 запросов/мин |

Заголовки ответа:
- `X-RateLimit-Limit` — максимум запросов
- `X-RateLimit-Remaining` — осталось запросов
- `X-RateLimit-Reset` — время сброса лимита

## Требования сетей

| Сеть | Подтверждения | Время блока |
|------|---------------|-------------|
| Tron | 20 | ~3 сек |
| Ethereum | 12 | ~12 сек |
| BSC | 15 | ~3 сек |

## Быстрый старт

<Steps>
  <Step title="Регистрация">
    Зарегистрируйте мерчанта через `/api/v1/merchants/register`
  </Step>
  <Step title="Сохраните ключи">
    Сохраните `apiKey` и `hmacSecret` из ответа
  </Step>
  <Step title="Создайте счёт">
    Создайте счёт через `/api/v1/invoices`
  </Step>
  <Step title="Настройте вебхук">
    Укажите URL для получения уведомлений об оплате
  </Step>
</Steps>
