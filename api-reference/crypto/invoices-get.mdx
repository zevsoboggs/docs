## Статусы счёта

| Статус | Описание |
|--------|----------|
| `Pending` | Ожидает оплаты |
| `PartiallyPaid` | Частичная оплата получена |
| `Paid` | Полностью оплачен |
| `Overpaid` | Получено больше требуемой суммы |
| `Expired` | Истёк срок оплаты |
| `Cancelled` | Отменён |
