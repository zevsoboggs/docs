## Получение баланса

```bash
curl -X GET "https://crypto.api.loveandpay.io/api/v1/withdrawals/balance" \
  -H "X-Api-Key: lc_ваш_ключ"
```

### Ответ

```json
{
  "success": true,
  "data": {
    "balances": [
      {
        "currency": "USDT_TRC20",
        "currencyName": "USDT_TRC20",
        "network": "Tron",
        "networkName": "Tron",
        "available": 1500.50,
        "pending": 100.00,
        "total": 1600.50
      },
      {
        "currency": "USDT_ERC20",
        "currencyName": "USDT_ERC20",
        "network": "Ethereum",
        "networkName": "Ethereum",
        "available": 500.00,
        "pending": 0,
        "total": 500.00
      }
    ],
    "totalUsdValue": 2100.50
  }
}
```

| Поле | Описание |
|------|----------|
| `available` | Доступно для вывода |
| `pending` | В ожидании подтверждения |
| `total` | Общий баланс |

## Создание вывода

<Note>
Требуется HMAC подпись. Комиссия за вывод: **1%**.
</Note>

```bash
curl -X POST "https://crypto.api.loveandpay.io/api/v1/withdrawals" \
  -H "Content-Type: application/json" \
  -H "X-Api-Key: lc_ваш_ключ" \
  -H "X-Timestamp: 1706400000" \
  -H "X-Signature: ваша_подпись" \
  -d '{
    "toAddress": "TQaqeXyS14drW5CB7HRo87yLPZjNvbGs1q",
    "amount": 100.00,
    "currency": "USDT_TRC20"
  }'
```

### Параметры

| Параметр | Тип | Обязательный | Описание |
|----------|-----|--------------|----------|
| `toAddress` | string | Да | Адрес кошелька получателя |
| `amount` | decimal | Да | Сумма вывода |
| `currency` | string | Да | Код валюты |

### Ответ

```json
{
  "success": true,
  "data": {
    "id": "ffe03a8b-3726-4227-963c-4136e0fa6213",
    "withdrawalNumber": "WD-20260121-ABC123",
    "toAddress": "TQaqeXyS14drW5CB7HRo87yLPZjNvbGs1q",
    "amount": 100.00,
    "fee": 1.00,
    "netAmount": 99.00,
    "currency": "USDT_TRC20",
    "network": "Tron",
    "status": "Pending",
    "createdAt": "2026-01-21T20:00:00Z"
  },
  "message": "Withdrawal request created successfully"
}
```

<Warning>
При выводе 100 USDT комиссия составит 1 USDT. Вы получите 99 USDT.
</Warning>

## Список выводов

```bash
curl -X GET "https://crypto.api.loveandpay.io/api/v1/withdrawals?page=1&status=Completed" \
  -H "X-Api-Key: lc_ваш_ключ"
```

### Query параметры

| Параметр | Тип | По умолчанию | Описание |
|----------|-----|--------------|----------|
| `page` | integer | 1 | Номер страницы |
| `pageSize` | integer | 20 | Размер страницы |
| `status` | string | — | Фильтр по статусу |

### Ответ

```json
{
  "success": true,
  "data": {
    "items": [
      {
        "id": "ffe03a8b-3726-4227-963c-4136e0fa6213",
        "withdrawalNumber": "WD-20260121-ABC123",
        "amount": 100.00,
        "fee": 1.00,
        "netAmount": 99.00,
        "status": "Completed",
        "txHash": "ce1a930bc5bbfcd1127174070dc7aaa0fadc1c8f712e59ba354d068830dc49bb"
      }
    ],
    "totalCount": 10,
    "page": 1,
    "pageSize": 20
  }
}
```

---

## Статусы вывода

| Статус | Описание |
|--------|----------|
| `Pending` | Запрос создан |
| `Processing` | Обрабатывается |
| `Completed` | Завершён |
| `Failed` | Ошибка |

---

## Ошибки

| Код | Описание |
|-----|----------|
| `INSUFFICIENT_BALANCE` | Недостаточно средств |
| `INVALID_ADDRESS` | Неверный адрес кошелька |
| `WITHDRAWAL_LIMIT_EXCEEDED` | Превышен лимит выводов |
