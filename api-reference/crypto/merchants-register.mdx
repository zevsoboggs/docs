<RequestExample>
```bash cURL
curl -X POST "https://crypto.api.loveandpay.io/api/v1/merchants/register" \
  -H "Content-Type: application/json" \
  -d '{
    "companyName": "My Company LLC",
    "email": "merchant@example.com",
    "address": "123 Main Street, Suite 100",
    "city": "New York",
    "country": "USA",
    "webhookUrl": "https://mysite.com/webhooks/crypto"
  }'
```

```python Python
import requests

response = requests.post(
    "https://crypto.api.loveandpay.io/api/v1/merchants/register",
    json={
        "companyName": "My Company LLC",
        "email": "merchant@example.com",
        "address": "123 Main Street, Suite 100",
        "city": "New York",
        "country": "USA",
        "webhookUrl": "https://mysite.com/webhooks/crypto"
    }
)
data = response.json()
print(f"API Key: {data['data']['apiKey']['apiKey']}")
print(f"HMAC Secret: {data['data']['apiKey']['hmacSecret']}")
```

```javascript JavaScript
const axios = require('axios');

axios.post('https://crypto.api.loveandpay.io/api/v1/merchants/register', {
    companyName: 'My Company LLC',
    email: 'merchant@example.com',
    address: '123 Main Street, Suite 100',
    city: 'New York',
    country: 'USA',
    webhookUrl: 'https://mysite.com/webhooks/crypto'
}).then(res => {
    console.log('API Key:', res.data.data.apiKey.apiKey);
    console.log('HMAC Secret:', res.data.data.apiKey.hmacSecret);
});
```
</RequestExample>

<Warning>
**Важно!** Сохраните `apiKey` и `hmacSecret` из ответа — они показываются **только один раз**!
</Warning>
