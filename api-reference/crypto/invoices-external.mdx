---
title: 'Счёт по External ID'
description: 'Получение счёта по вашему внешнему идентификатору'
api: 'GET https://crypto.api.loveandpay.io/api/v1/invoices/external/{externalId}'
---

## Параметры пути

<ParamField path="externalId" type="string" required>
  Ваш внешний ID заказа, например `order-12345`
</ParamField>

## Заголовки

<ParamField header="X-Api-Key" type="string" required>
  Ваш API ключ
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://crypto.api.loveandpay.io/api/v1/invoices/external/order-12345" \
  -H "X-Api-Key: lc_ваш_ключ"
```

```python Python
import requests

response = requests.get(
    "https://crypto.api.loveandpay.io/api/v1/invoices/external/order-12345",
    headers={"X-Api-Key": "lc_ваш_ключ"}
)
print(response.json())
```

```javascript JavaScript
const axios = require('axios');

axios.get('https://crypto.api.loveandpay.io/api/v1/invoices/external/order-12345', {
    headers: { 'X-Api-Key': 'lc_ваш_ключ' }
}).then(res => console.log(res.data));
```
