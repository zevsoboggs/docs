## Получение статуса KYC сессии

Возвращает текущий статус и детали сессии верификации.

### Аутентификация

<ParamField header="x-api-key" type="string" required>
  Ваш API ключ
</ParamField>

### Path параметры

<ParamField path="id" type="string" required>
  Уникальный идентификатор KYC сессии (например: `kyc_abc123def456`)
</ParamField>

### Пример запроса

<CodeGroup>

```bash cURL
curl -X GET "https://loveandpay.io/api/v1/kyc/sessions/kyc_abc123def456" \
  -H "x-api-key: YOUR_API_KEY"
```

```javascript Node.js
const sessionId = 'kyc_abc123def456';

const response = await fetch(
  `https://loveandpay.io/api/v1/kyc/sessions/${sessionId}`,
  {
    headers: {
      'x-api-key': process.env.LOVEANDPAY_API_KEY
    }
  }
);

const session = await response.json();

if (session.status === 'completed') {
  console.log('Верификация пройдена!');
  console.log('Данные:', session.verifiedData);
} else if (session.status === 'failed') {
  console.log('Причина отказа:', session.failureReason);
}
```

```python Python
import requests

session_id = 'kyc_abc123def456'

response = requests.get(
    f'https://loveandpay.io/api/v1/kyc/sessions/{session_id}',
    headers={'x-api-key': 'YOUR_API_KEY'}
)

session = response.json()

if session['status'] == 'completed':
    print('Верификация пройдена!')
    print('Данные:', session['verifiedData'])
elif session['status'] == 'failed':
    print('Причина отказа:', session['failureReason'])
```

</CodeGroup>

## Статусы сессии

| Статус | Описание |
|--------|----------|
| `pending` | Сессия создана, ожидает начала верификации |
| `in_progress` | Пользователь проходит верификацию |
| `completed` | Верификация успешно завершена |
| `failed` | Верификация не пройдена |
| `expired` | Сессия истекла (не использована в течение 24 часов) |

## Причины отказа

| Код | Описание |
|-----|----------|
| `document_expired` | Срок действия документа истёк |
| `document_unreadable` | Документ нечитаем или плохого качества |
| `document_mismatch` | Данные документа не совпадают |
| `selfie_mismatch` | Лицо на селфи не совпадает с фото в документе |
| `suspected_fraud` | Подозрение на мошенничество |
| `unsupported_document` | Тип документа не поддерживается |

### Ошибки

| Код | Описание |
|-----|----------|
| 401 | Неверный API ключ |
| 404 | Сессия не найдена |
