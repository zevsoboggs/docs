<Warning>
**Создание вебхуков через публичный API отключено.**

Это политика безопасности — чтобы скомпрометированный API-ключ не позволил перенаправить ваши финансовые уведомления на чужой сервер.
</Warning>

## Как создать

Через [Панель управления → Вебхуки](https://loveandpay.io/dashboard/webhooks).

Полное руководство — [Вебхуки](/guides/webhooks).
