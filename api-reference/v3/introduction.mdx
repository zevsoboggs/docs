<Note>
**API v3 находится в активной разработке.** Эта страница — предварительный анонс.
Эндпоинты, форматы запросов/ответов и сроки могут измениться. Для production
используйте [API v2](/api-reference/v2/introduction).
</Note>

## Совместимость

- **API v2 остаётся основным** и поддерживается. Переход на v3 будет добровольным,
  с переходным периодом и инструкцией по миграции.
- Подпись запросов (HMAC-SHA256) и принципы аутентификации сохранятся —
  см. [Аутентификация v2](/api-reference/v2/authentication).
