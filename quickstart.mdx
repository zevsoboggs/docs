## Начните за три шага

Запустите сайт документации локально и сделайте первую настройку.

### Шаг 1: Настройте локальное окружение

<AccordionGroup>
  <Accordion icon="copy" title="Клонируйте документацию локально">
    Во время регистрации вы создали GitHub репозиторий с контентом документации, если у вас его ещё не было. Ссылку на репозиторий можно найти в [панели управления](https://dashboard.mintlify.com).

    Чтобы клонировать репозиторий локально для внесения и предпросмотра изменений, следуйте руководству [Клонирование репозитория](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) в документации GitHub.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Запустите сервер предпросмотра">
    1. Установите Mintlify CLI: `npm i -g mint`
    2. Перейдите в папку с документацией и выполните: `mint dev`
    3. Откройте `http://localhost:3000` чтобы увидеть документацию!

    <Tip>Предпросмотр обновляется автоматически при редактировании файлов.</Tip>
  </Accordion>
</AccordionGroup>

### Шаг 2: Разверните изменения

<AccordionGroup>
  <Accordion icon="github" title="Установите наше GitHub приложение">
    Установите приложение Mintlify GitHub из [панели управления](https://dashboard.mintlify.com/settings/organization/github-app).

    Наше GitHub приложение автоматически разворачивает изменения на сайте документации, так что вам не нужно управлять деплоем самостоятельно.
</Accordion>
<Accordion icon="palette" title="Обновите название и цвета сайта">
    Для первого изменения давайте обновим название и цвета сайта документации.

    1. Откройте `docs.json` в редакторе.
    2. Измените поле `"name"` на название вашего проекта.
    3. Обновите `"colors"` под ваш бренд.
    4. Сохраните и сразу увидите изменения на `http://localhost:3000`.

    <Tip>Попробуйте изменить основной цвет, чтобы сразу увидеть разницу!</Tip>
  </Accordion>
</AccordionGroup>

### Шаг 3: Запуск в продакшн

<Accordion icon="rocket" title="Опубликуйте документацию">
  1. Сделайте коммит и запушьте изменения.
  2. Ваша документация обновится и будет доступна через несколько мгновений!
</Accordion>
